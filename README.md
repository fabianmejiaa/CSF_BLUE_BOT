# Asistente Virtual con Voz

Bot web que habla y escucha usando la Web Speech API.

## Características
- Reconocimiento de voz (te escucha)
- Síntesis de voz (te habla)
- Interfaz moderna con animaciones
- Historial local
- Totalmente gratuito (sin APIs externas)

## Uso
1. Abrir en Chrome/Edge (soporta voz)
2. Permitir acceso al micrófono
3. Hablar o escribir
4. El bot responde con voz y texto
